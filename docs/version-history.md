### developer notes 0 (9 Jul 2022)
Hello world. The aim of building this website is to try out the integration of a ML network with a web app while trying out development best practices. In the few days, I will add a spec that describes the requirements of the web app.

Meanwhile, here are the best practices that I would try to emulate:
- following a spec sheet
- CI/CD
- automated testing where possible
- automated building
- writing to different branch and testing before merging to master

So far, I have a rough roadmap of how to build this website:
- learn how to write a basic ML system
- learn how to incorporate external ML system into back end
- learn how to use docker /and kubernetes
- learn how to deploy on the web
- build the frontend, backend and ML model
- deploy website

I have not decided on which backend or ML model to use, but I am pretty much settled on React as my frontend.
