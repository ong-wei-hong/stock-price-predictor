# Spec

## Features
### Website features
 - Users should be able to request a prediction by submitting a ticker irregardless of if they are logged in or not
 - Users should be able to see past predictions made by the website
 - Users should be able to see how accurate the website is (global correct/total predictions)
 - Users should be able to search up other users
 - Users should be able to see how accurate the website is for searched users
 - Users should be able to see user past predictions
### User features
 - Users should be able to create an account
 - Users should be able to log in
 - Users should be able to follow other users
 - Users should be able to see who they are following
 - Users should be able to see who follows them
 - Users should be able to see past requested predictions
 - Users should be able to see past requested predictions of themselves and who they follow in one screen
 - Users should be able to see past predictions of other users
 - Users should be able to reset password through email
 - Users should be able to set multiple emails
 - Users should be able to add emails
 - Users should be able to delete emails as long as they have at least 1 email
 - Users should be able to set a profile pic
### Misc features
 - Site should have a privacy policy for emails
 - Site should have a disclaimer that website is for educational purposes
